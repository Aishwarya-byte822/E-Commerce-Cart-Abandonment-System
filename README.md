# E-Commerce Cart Abandonment Dashboard

## Overview
This project analyzes customer behavior in an e-commerce platform and identifies cart abandonment trends.

## Features
- KPI Dashboard
- Revenue Analysis
- Cart Abandonment Analysis
- Conversion Funnel
- Top Brands
- Top Products
- Active Users
- Interactive Filters

## Tech Stack
- Python
- SQLite
- Pandas
- SQLAlchemy
- Streamlit

## Run Locally

pip install -r requirements.txt

streamlit run dashboard.py
